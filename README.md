# The Islamic Timeline

Live: https://aminehamlouchi.github.io/timeline/

An interactive vertical timeline from today back through the prophets to Adam (as), and out to ancient and world history at true scale. Atlas borders from Historical Basemaps (Ourednik, GPL-3.0). Built with Next.js (static export).

Suggest a record via the in-app button. Support: https://cash.app/$IbnHamlouchi
